## 💡 Sobre o projeto

Este é um projeto aberto que serve como alternativa ao Linktree website.
- [Demo](https://joaobosco2011.github.io/)

## 🚀 Clonando o Repositório

Se tiver o git instalado em seu computador, pode clonar o repositório para o seu pc ou baixar um arquivo ZIP com todos os arquivos diretamente.
[Baixe o ZIP desse local diretamente](https://github.com/joaobosco2011/joaobosco2011.github.io/archive/refs/heads/main.zip), ou rode o seguinte comando para clonar os arquivos para o seu pc:
```bash
git clone https://github.com/joaobosco2011/joaobosco2011.github.io
```
- Uma vez que os arquivos estejam em seu pc, abra a pasta do Linktree no [Visual Studio Code](https://code.visualstudio.com/).
- Com os arquivos abertos no Visual Studio Code, clique no botão **Go Live** na parte inferior da janela do VS Code para carregar os arquivos com o [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).
- Altere a imagem de perfil e os textos no arquivo `index.html`.
- Altere as cores e fontes no arquivo `style.css`.

## 🤔 Como Contribuir

- Faça um Fork do repositório;
- Crie uma branch com suas modificações: `git checkout -b my-feature`;
- Commit suas alterações: `git commit -m "feat: my new feature"`;
- Push para a sua branch: `git push origin my-feature`.

Uma vez que o seu pull request receba um merged, você pode deletar a sua branch.
